Un template avec des environnements (théorèmes, définition, etc.) pour rédiger des notes de cours de mathématique. Le style des environnements peut être personnalisé via des set-rules.
Basé sur les packages typst suivants:
- showybox (https://github.com/Pablo-Gonzalez-Calderon/showybox-package)
- elembic (https://github.com/PgBiel/elembic)
- rich-counters (https://github.com/jbirnick/typst-rich-counters)
